# Lost Atelier - Propuesta de Consultoría | PRD

## Problema Original
Crear una landing page premium y editorial para presentar el presupuesto de consultoría estratégica de WTF Agency para Lost Atelier (empresa de muebles artesanales de barricas de vino de Mendoza, Argentina).

## User Persona
- **Cliente objetivo:** Lost Atelier (empresa B2B)
- **Presentador:** WTF Agency
- **Propósito:** Mostrar propuesta de consultoría de USD 12,000 por 45 días de trabajo

## Requisitos Core
- Landing informativa sin funcionalidades adicionales
- Estética editorial premium con imágenes de impacto
- Tipografía serif elegante (Playfair Display)
- Paleta oscura: negro, carbón, borgoña/vino, dorado

## Lo Implementado (Diciembre 2025)
- ✅ Hero section con imagen de barricas de vino y smooth scroll
- ✅ Sección Diagnóstico - "El Problema" de comunicación
- ✅ Sección Propuesta de Valor - "Artesanía con Alma"
- ✅ Sección Metodología - 3 Fases (45 días)
- ✅ Sección Inversión - USD 12,000
- ✅ Sección Entregables - 9 items detallados
- ✅ Sección Contacto con CTAs por email
- ✅ Smooth scroll con Lenis
- ✅ Animaciones con Framer Motion
- ✅ Diseño responsive completo

## Stack Técnico
- Frontend: React + Tailwind CSS
- Animaciones: Framer Motion + Lenis
- Tipografía: Playfair Display (headings) + Manrope (body)
